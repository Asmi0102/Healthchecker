🧬 BioHealth Assistant

BioHealth Assistant is a lightweight, browser-based health dashboard built using HTML, CSS, and JavaScript.
It provides interactive tools for basic health insights including BMI calculation, blood group compatibility checking, and a disease symptom predictor.

The project focuses on clean UI design, simple health tools, and a responsive user experience with dark/light theme support.

🚀 Features
📊 BMI Calculator

Calculate Body Mass Index (BMI) using height and weight inputs.

Includes

Real-time BMI calculation
BMI category classification
Simple health interpretation
🩸 Blood Group Compatibility Checker

Check blood donation compatibility between donor and receiver blood groups.

Functionality

Select donor blood group
Select receiver blood group
Instant compatibility result
Easy-to-understand feedback
🩺 Disease Symptom Checker

A heuristic-based disease suggestion tool.

Users can:

Select symptoms
Receive possible disease suggestions
View recommended medicines
Get basic health advice and tips

⚠️ This feature is intended for educational purposes only and should not replace professional medical consultation.

🌙 Dark / Light Mode

The application supports theme switching.

Features include:

Dark mode toggle in navbar
Theme preference saved using localStorage
Smooth theme switching
Accessible toggle button with labels

Theme icons:

🌙 Dark Mode
☀️ Light Mode
🧭 Scroll-Aware Navigation

Navigation links automatically highlight based on the section currently visible on screen.

This improves:

User navigation
Page readability
Overall UI experience
